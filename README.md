# Power BI Portfolio

This repository showcases Power BI dashboards and reporting solutions that I have designed to support clinical operations, quality improvement, and research analytics within academic healthcare.

My work spans the complete analytics lifecycle—from gathering stakeholder requirements and designing SQL queries to data engineering, data modeling, DAX development, visualization design, automation, and dashboard deployment.

A recurring challenge in healthcare analytics is that data does not always originate from a single enterprise data warehouse. Many projects require integrating information from multiple clinical systems, research databases, and institutional environments, each with unique database structures, business rules, and reporting requirements. The projects in this repository demonstrate approaches to designing scalable reporting solutions across these distributed data sources.

> **Note:** All examples use fictional, de-identified, or recreated datasets. Proprietary institutional logic and protected health information (PHI) have been removed.

## Anesthesia Quality Improvement Dashboard

## Adolescent & Young Adult (AYA) Cancer Dashboard

Designed and developed enterprise dashboards supporting an Adolescent and Young Adult Cancer Program across two independent healthcare organizations.

### Project Highlights

- Developed independent SQL reporting pipelines for each health system
- Navigated differences in database structures, workflows, and reporting logic across enterprise environments
- Integrated cancer registry, treatment, referral, survivorship, and demographic data
- Developed patient matching and deduplication logic to identify patients receiving care across multiple organizations
- Built a consolidated reporting model while preserving institution-specific reporting requirements
- Captured cross-institution treatment activity, including patients diagnosed at one organization who received radiation therapy at another
- Created interactive dashboards supporting clinicians, researchers, and program leadership

### Technologies

- PostgreSQL
- SQL
- Power BI
- DAX
- Power Query
